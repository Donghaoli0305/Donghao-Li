Project Overview

This project focuses on analyzing COVID-19-related tweets to extract meaningful insights, such as sentiment trends, hashtag usage, and pandemic-related hotspots. The system leverages Natural Language Processing (NLP) techniques and sentiment analysis to provide a deeper understanding of social media data during global crises.



Prerequisites

Before installing and running the system, ensure you have the following:

Python 3.8 or higher

An internet connection to download NLTK resources

CSV files containing COVID-19-related tweets in the specified format


You can install the required libraries using pip:

pip install pandas

pip install nltk
